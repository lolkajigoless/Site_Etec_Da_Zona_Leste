## Como rodar o projeto:

1. Instalar Laravel (consultar documentação), preferencialmente a v.13(mais recente) que é usada neste projeto;
2. Instalar o NPM: Entre no site do NodeJs e baixe-o (NPM ja vem junto com o NodeJs);
3. Estando com o Laravel instalado, execute o comando: `php artisan serve`;
4. Estando com o NPM instalado, execute o comando: `npm run dev`, é nescessario para o Tailwind poder funcionar;
5. Entre no link que o `php artisan serve` dá quando for executado e você ja poderá usar o site.
